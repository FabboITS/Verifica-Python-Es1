# Passgen

Progetto Git Python per generare password casuali.

## Struttura

- `ex01/Passgen.py`: programma principale che genera una password semplice o complessa.

## Uso

Esegui il programma con Python:

```bash
python3 ex01/Passgen.py
```

Scegli `1` per ottenere una password semplice di 8 caratteri alfanumerici.
Scegli `2` per ottenere una password complessa di 20 caratteri ASCII.

## Debug

Il codice contiene messaggi di debug che mostrano l'avanzamento della generazione della password e l'input utente.
